SeedMe's data sharing sandbox builder

This page creates a free temporary website with which you can safely try SeedMe's data sharing modules for Drupal 8.

How to build a sandbox website:

  1. Click the Create button below to create your site.
  2. Write down the demo account password.
  3. Click on the Sandbox website link below.
  4. Log in with the demo account to try out data sharing.
  5. Browse help info on the site.

The sandbox will be automatically deleted after 7 days.
